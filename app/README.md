# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1BHAdM4NvJbMdB_C2BMRIj4d9tvk5KDpP

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `DEEPSEEK1_API_KEY` (or `DEEPSEEK_API_KEY`) in [.env.local](.env.local) to your DeepSeek API key
3. Run the app:
   `npm run dev`
